# SQL Server & Data Engineering Portfolio

Portfolio técnico frontend creado con React, Vite y Tailwind CSS para mostrar conocimientos de SQL Server, ETL, reporterías y data engineering mediante ejemplos ficticios inspirados en escenarios reales.

## Stack

- React
- Vite
- Tailwind CSS
- JavaScript
- Lucide React

## Ejecutar localmente

```bash
npm install
npm run dev
```

Luego abre la URL local que muestra Vite en la terminal.

## Estructura

```text
src/
  components/
  data/
  App.jsx
  main.jsx
  index.css
```

Para agregar más técnicas SQL o casos prácticos, edita los arreglos en:

- `src/data/sqlTechniques.js`
- `src/data/caseStudies.js`
